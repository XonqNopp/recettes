Gratin de pâtes
###############

* Préparation: 30 min
* Cuisson: 30 min


Ingrédients
===========

+------------+-------------+----------------------------------------------------+
| 1 personne | 6 personnes |                                                    |
+============+=============+====================================================+
|       80 g |       500 g | pâtes                                              |
+------------+-------------+----------------------------------------------------+
|        1/3 |           2 | bocaux de sauce tomates                            |
+------------+-------------+----------------------------------------------------+
|            |             | lardons                                            |
+------------+-------------+----------------------------------------------------+
|            |             | jambon                                             |
+------------+-------------+----------------------------------------------------+
|            |             | demi-crème                                         |
+------------+-------------+----------------------------------------------------+
|            |             | fromage râpé                                       |
+------------+-------------+----------------------------------------------------+


Préparation
===========

* Cuire les pâtes (dans plusieurs casseroles si nécessaire)
* Tout mettre dans un plat à gratin.
* Environ 30min au four à 200°C.
